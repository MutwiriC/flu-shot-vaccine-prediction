# Flu Vaccine Prediction - Phase 3 Project

## Project Overview

Predicting seasonal flu vaccine uptake using demographic and behavioral survey data.

## Business Problem

The CDC wants to increase flu vaccination rates by identifying populations unlikely to vaccinate.

## Data

- Source: DrivenData Flu Shot Learning Competition
- Training Features: `training_set_features.csv`
- Training Labels: `training_set_labels.csv`
- Test Features: `test_set_features.csv`
